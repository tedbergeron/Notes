AsciiDoc is like a superset of Markdown. 

=== Chrome Extention 

Get the extension for Chrome *Asciidoctor Live Preview*: https://chrome.google.com/webstore/detail/asciidoctorjs-live-previe/iaalpfgpbocpdfblpnhhgllgbdbchmia?hl=en-US

* Set Options
** Safe mode: unsafe (so you can use experimental things such as showing the keyboard keys in your docs)
** Preview .txt files [x] (check so you can use .txt as well as .adoc files)

Save the test file and open in Chrome.

=== Local AsciiDoc Website
Create a local website on your laptop with text files and Chrome: Asciidoctor Live Preview extension. This post has some template text files to download to get you started with the wiki-like website. http://tedbergeron.github.io/2015/04/09/AsciiDoc-Website.html
